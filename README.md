# Automatic Night Light Using ATmega32

## Project Overview
This project implements an automatic night light system using the ATmega32 microcontroller, LDR sensor, ADC, and LED.
The system automatically turns ON the LED in darkness and turns it OFF in bright light.

---

## Components Used
- ATmega32
- LDR
- 10kΩ Resistor
- LED
- ground
- capacitor (10nf)
- power supply(5v)
- 220Ω Resistor
- SimulIDE
- Microchip Studio

---

## Features
- Automatic light detection
- ADC-based analog input processing
- Low power embedded system
- Assembly language implementation
